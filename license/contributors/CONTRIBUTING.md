# Contributing

We welcome contributions! However, to ensure consistency and quality:

- Paid [Repository Access](../access/REPOSITORY-ACCESS.md) is required before you can clone or open pull requests.
- All changes must be submitted as **pull requests (PRs)** to this repository.
- Forks may exist for personal use or experimentation, but only this repository is the official source of truth. See [NOTICE.md](../NOTICE.md).
- By contributing, you agree that your changes will be governed by the same license and included in the upstream project.
- Significant contributions may require signing the [Contributor License Agreement (CLA)](CLA.md).
