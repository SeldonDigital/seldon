# Values

This folder holds per-property wire shapes and `PropertySchema` modules. Each property key exports typed cells, optional enums, and a schema entry that feeds the flattened catalog in `schemas/data/property-schemas.ts`. Folder layout does not always match inspector category order. See `values/index.ts` and the [properties README](../README.md) for the canonical catalog sequence.

---

## Flow

```mermaid
flowchart LR
  author[Authoring cell] --> module[values module]
  module --> catalog[PROPERTY_SCHEMAS]
  catalog --> validate[validatePropertyValue]
```
